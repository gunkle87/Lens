# Lens Pillar Tracker

This file is a template and guiderail for pillar execution tracking.

Do not treat this file as the live execution record for every task and pillar.
For active execution, copy this template into a run-specific tracker file and
update that file instead.

Use it with `BUILD_PROTOCOL.md`.

## Global Gates

- [ ] Pre-task verification passed for current task
- [ ] All prior tasks are fully complete (implementation + audit + commit)
- [ ] No unresolved blockers or non-blockers remain
- [ ] Canonical test runner is available and used exclusively
